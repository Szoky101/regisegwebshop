# Régiség Webshop

A **Régiség Webshop** egy React + Laravel alapú webes alkalmazás, amelynek célja, hogy összekösse a régiségek iránt érdeklődő embereket. A platform lehetőséget biztosít antik tárgyak feltöltésére, böngészésére, adásvételére és kommunikációra.

## 📦 Technológiák
- **Frontend:** React, Tailwind CSS
- **Backend:** Laravel
- **Adatbázis:** MySQL / MariaDB
- **API Kommunikáció:** REST API

## 🎯 Projekt Célja
- Egy olyan közösségi alapú piactér létrehozása, ahol a felhasználók:
  - Régiségeket adhatnak el
  - Ritkaságokra kereshetnek
  - Kapcsolatba léphetnek egymással
  - És egy kimondott régiség kánaán létrehozása

## ✨ Fő Funkciók
- 🔐 Felhasználói regisztráció és bejelentkezés  
- 🛒 Termékek feltöltése, szerkesztése, törlése  
- 🔍 Keresés és szűrés kategóriák alapján  
- 💬 Üzenetküldés eladók és vásárlók között  
- ⭐ Kedvencek listája  
- 📜 Részletes termékoldal
